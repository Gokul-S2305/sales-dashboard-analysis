# 📊 Sales Performance Dashboard

## 🚀 Project Overview

This project analyzes sales performance using Python and Tableau to provide meaningful business insights through interactive visualizations and dashboards.

The objective is to help stakeholders understand sales trends, customer behavior, product performance, and regional sales distribution for better decision-making.

---

## 🎯 Project Objectives

- Analyze historical sales data.
- Identify top-performing products and customers.
- Evaluate sales performance across countries and territories.
- Track monthly sales trends.
- Create an interactive dashboard for business intelligence.
- Present insights in a visually appealing and user-friendly format.

---

## 📈 Key Performance Indicators (KPIs)

The dashboard includes:

- Total Sales
- Total Orders
- Total Customers
- Average Order Value (AOV)

---

## 📊 Dashboard Features

### 1. Monthly Sales Trend
Visualizes sales performance over time to identify growth patterns and seasonal trends.

### 2. Sales by Product Line
Compares revenue generated by different product categories.

### 3. Top Customers Analysis
Highlights customers contributing the highest revenue.

### 4. Country Performance
Displays geographical distribution of sales using an interactive map.

### 5. Territory Performance
Analyzes sales contribution from different territories.

### 6. Deal Size Distribution
Shows the proportion of Small, Medium, and Large deals.

### 7. Interactive Filters
Users can filter dashboard results by:
- Country
- Product Line
- Territory
- Deal Size
- Year
